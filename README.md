# Cybersecurity Lab Portfolio

This repository contains hands-on cybersecurity lab documentation demonstrating practical skills in network security, threat detection, vulnerability management, incident response, and security operations.

The labs were completed in controlled environments using structured lab simulations and independent practice setups. Each lab is documented with objectives, configuration steps, evidence, results, and real-world security relevance.
